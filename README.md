# Superstore Sales Data Analysis
This repository contains my exploratory data analysis (EDA) project on the Sample Superstore dataset.

The purpose of this project was to practice data analysis using Python and improve my understanding of Pandas, NumPy, Matplotlib, and Seaborn by working on a real dataset.

## What I Worked On
-  loaded the dataset on jupyter lab
-  Explorerd the datset
- Checked if there is any kind of duplicate values
- Converted date columns into datetime format
-  Created new features such as order year, month, and shipping time.
- Analyzed the profit and sales
- Analyzed segment wise and region wise profit and sales
- Created visualizations using Matplotlib and Seaborn to gain business insights.
